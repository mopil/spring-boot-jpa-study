요구사항: 프로그램 실행을 위해 JRE가 설치되어 있어야 합니다.

화면 모니터링 프로그램 다운로드 링크:
https://www.dropbox.com/sh/nel4975znuwactx/AAChxvl3zdoqioFd0op-04Kua?dl=0


[Windows 환경 실행 순서]
1. 위 다운로드 링크에서 monitor2.3.exe 파일을 다운로드 받습니다.
2. 다운 받은 monitor2.3.exe 파일을 실행합니다.
   - 만약 사용 중인 PC에 JRE가 설치되어 있지 않은 경우 실행되지 않습니다.
     그런 경우 https://java.com/ko/download/ 링크에서 설치 파일을 다운 받아 우선 JRE를 설치한 후
     monitor2.3.exe 파일을 다시 실행합니다.
3. monitor2.3.exe 파일을 실행하면 PC 화면 왼쪽상단에 창이 나타나며 다음과 같이 입력합니다.
   - 주소: 117.17.158.46   ID: (본인학번)   이름: (한글이름)
4. 입력 후 "확인" 버튼을 클릭합니다.


[Mac 환경 실행 순서]
1. 위 다운로드 링크에서 monitor2.3.jar 파일을 다운로드 받습니다.
2. 터미널에서 다음 명령 실행을 통해 다운 받은 monitor2.3.jar을 실행합니다.
     java -jar monitor2.3.jar

   Mac의 경우 반드시 터미널에서 위 명령 실행을 통해 monitor2.3.jar을 실행해야 합니다.
   그렇지 않을 경우 문제가 생길 수 있습니다.
(나머지는 Windows 환경 실행 순서와 동일합니다)
